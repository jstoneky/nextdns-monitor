Console Errors

If the extension is installed but broken, check the background page console:

Chrome: Go to chrome://extensions → find DNS Medic → click "Inspect views: background page" → Console tab → copy any red errors.

Firefox: Go to about:debugging → This Firefox → DNS Medic → click Inspect → Console tab → copy any red errors.
